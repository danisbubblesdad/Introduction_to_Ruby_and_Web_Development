5. There are a number of variables listed below. What are the different types
and how do you know which is which?

excited_dog = "excited dog"
This is a local variable since there are no symbols prefixing the variable
name.

@excited_dog = "excited dog"
This is an instance variable since there is an @ symbol prefix.

@@excited_dog = "excited dog"
This is a class variable since there is the "@@" prefix.
